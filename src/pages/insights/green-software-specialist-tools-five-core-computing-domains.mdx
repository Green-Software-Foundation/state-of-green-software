There are five core domains of computing addressed by specialist green software tools, each at different levels of maturity. The core domains include [AI](https://www.techtarget.com/searchenterpriseai/feature/Green-AI-tackles-effects-of-AI-ML-on-climate-change), [code](https://www.ibm.com/cloud/blog/green-coding), [operating system](https://www.redalyc.org/pdf/2652/265219635005.pdf)s, [cloud,](https://www.accenture.com/us-en/insights/strategy/green-behind-cloud) and[ web](https://www.thegreenwebfoundation.org/).
![Five core computing domains](/images/insights/27-additional-1.png)
These domains are not mutually exclusive and software practitioners should use a combination to achieve the most accurate results and optimum impact on carbon emissions.
